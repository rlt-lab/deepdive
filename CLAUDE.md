# Deepdive

Turn-based roguelike dungeon crawler built with Rust and Bevy 0.16.1.

## Build & Run

```bash
cargo run --release
```

## Project Structure

- `src/` - Rust source (main.rs has system organization)
- `assets/` - Sprites (32x32), config (.ron), fonts
- `docs/` - Extended documentation
